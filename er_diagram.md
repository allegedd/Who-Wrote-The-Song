### 本サービスの概要（700 文字以内）

ユーザが入力した曲の作曲家、作詞家を調べます。  
その作曲家、作詞家関わった他の楽曲を発見できるサービスです。

### MVP で実装する予定の機能

- [x] ユーザー登録機能
- [x] ログイン機能
- [x] パスワード変更機能
- [x] 楽曲検索機能（未ログインでも操作可能）
- [x] 楽曲プレビュー（未ログインでも操作可能）
- [x] お気に入り機能
- [x] お気に入り解除機能
